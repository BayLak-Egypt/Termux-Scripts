🔐 Password Hash Cracker

A unified Python-based security tool that combines password collection, hash analysis, and smart caching to improve performance.

The tool is designed to:

Download and maintain a password list without duplicates

Analyze hashes using multiple algorithms

Cache previously checked hashes to avoid re-processing

📁 Project Structure

project/
│
├── main.py        # Main tool (password fetching + hash analysis)
├── passwords.txt  # Password wordlist (auto-generated & updated)
├── hashes.bin     # Local binary database for checked hashes
└── README.md      # Documentation


---

⚙️ Requirements

Python 3.8+

Internet connection (for password fetching)


If external libraries are used in the code:

pip install -r requirements.txt


---

🧠 How It Works

The entire logic is handled inside main.py:

Fetches passwords from an online source

Stores them in passwords.txt while preventing duplicates

Uses the password list to analyze hashes

Saves already-checked hashes into hashes.bin

Skips hashes that were previously analyzed to significantly improve speed



---

🗄️ hashes.bin (Hash Cache)

hashes.bin acts as a local binary cache:

Stores hashes that have already been processed

Prevents redundant checks on subsequent runs

Greatly improves performance for repeated executions


> ⚠️ Do NOT edit this file manually.




---

🔑 Usage

python main.py -a ALGO [-H HASH]

Command-Line Options

Option	Description

-h	Show help message
-a	Hash algorithm (required)
-H	Target hash to analyze (optional)



---

🧪 Usage Examples

Run using only an algorithm:

python main.py -a md5

Run with a specific hash:

python main.py -a sha256 -H 5e884898da28047151d0e56f8dc62927
