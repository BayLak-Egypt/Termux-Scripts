## 📟 Project Overview
**Termux-Scripts** is a versatile toolkit designed for mobile penetration testers and Linux enthusiasts. It streamlines the process of installing, configuring, and executing security tools directly from your Android device.

---

## 🚀 Key Features
* 🔍 **OSINT & Recon:** Automated tools for information gathering.
* 🛡️ **Security Auditing:** Scripts for vulnerability scanning and network analysis.
* ⚡ **One-Tap Setup:** Dependency installer to get you started in seconds.
* 🎨 **Custom UI:** Clean, color-coded terminal interface for better readability.

---

## 🛠️ Prerequisites
Before running the scripts, ensure your Termux environment is up to date:

```pkg update && pkg upgrade -y```

```pkg install git python php curl -y```
```git clone https://github.com/BayLak-Egypt/Termux-Scripts.git```
```cd Termux-Scripts```
